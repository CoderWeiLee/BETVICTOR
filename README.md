## Introduce
-  BETVICTOR伟德由成立于马恩岛的MayFly娱乐有限公司掌舵，拥有马恩岛政府所发执照。
- 1946年始创于英国的BETVICTOR，是世界领先的网络博彩品牌之一，70年来不余遗力为全球玩家提供高效、卓越与安全的博彩产品，并以高端的服务品质获得广泛认可，带来最佳的在线娱乐体验。
-  英超利物浦足球俱乐部主要赞助商
